This is a simple website to keep track of the books we read.
Its just simple as maintaining notes.

1) clone the git repository
2) run "npm install" to install all the node_modules
3) just run "node server.js"
<img width="1919" height="1031" alt="image" src="https://github.com/user-attachments/assets/47a48d31-0983-49e4-a5e6-84a4141eea0c" />

4) And now open your "http://localhost:3000"
5) You can view the user interface of Notely website
6) Press the "+" button on the bottom right corner
<img width="1891" height="919" alt="image" src="https://github.com/user-attachments/assets/f302f29d-ccec-4d17-90e7-4abea76112df" />

7) Add the book you read with short description
<img width="1895" height="919" alt="image" src="https://github.com/user-attachments/assets/96fae0c9-273f-4388-8a11-6b3897316a07" />

8) Now your data is stored in your local system under the location "/Notely/static/notes.json"
<img width="1871" height="994" alt="image" src="https://github.com/user-attachments/assets/741633cf-9cd2-438c-8a92-9ce8ada89a1d" />

9) so whenever you relaunch the website your old data will be preserved
10) :)
